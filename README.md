# Environmental Gerontology – Community Services and Rehabilitation
This repository contains the source code for a website about environmental gerontology, focusing on community services, rehabilitation, and supportive environments for older adults. 
The project presents concepts, services, and knowledge resources aimed at promoting healthy aging, independence, and community-based support.

## Overview
This repository contains the source code for a website focused on **environmental gerontology**, community-based services, and rehabilitation for older adults.  
The project presents knowledge, services, and concepts that support healthy aging, independence, and the integration of healthcare and community environments.

The website introduces key domains such as environmental gerontology, gerontechnology, and rehabilitation, and demonstrates how supportive environments can improve quality of life for older adults.

## Features
- Introduction to **Environmental Gerontology**
- Overview of **community services and aging-friendly environments**
- Presentation of **rehabilitation and support services**
- Knowledge resources for professionals working with older adults
- Clean and minimal visual design with soft illustrations

## Structure
/assets → images and illustrations
/styles → CSS styles
/scripts → JavaScript files
index.html → main landing page
README.md → project documentation

## Purpose
The goal of this project is to communicate how **community infrastructure, healthcare services, and supportive environments** can work together to enable aging with dignity, independence, and well-being.
